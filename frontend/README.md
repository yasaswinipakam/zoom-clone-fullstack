# Zoom Clone — Frontend

Next.js + React + TypeScript + Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

Dev server runs at `http://localhost:3000`.

Backend must be running at `http://localhost:8000`.

## Environment

Create `.env.local`:

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```
